# ImprovedTalents-WotLK

Improved Talents panel for WoW WotLK 3.3.5a client

# Description

Improved Talents is a replacement talent frame addon for WoW 3.3.5 (WotLK). Instead of the default Blizzard talent UI, it displays all three talent trees side by side in a single wide panel, eliminating the need to switch tabs.

# Features

  - Side-by-side trees - all three talent trees visible simultaneously in one frame, scaled to fit
  - Dynamic layout - frame width adjusts automatically to the number of talent tabs (1–3); works for all classes
  - Dual specialization - spec tab buttons (top-left) styled after the original WoW spec tabs; icon shows the primary
  tree of each spec; gold glow on the currently active spec; gold overlay on the currently viewed spec; "Set Active"
  button appears when viewing the inactive spec
  - Preview mode - "Preview" checkbox lets you tentatively stage talent points before spending them;
  left-click stages a point, right-click unstages; tree headers reflect staged + real points; a Learn/Reset bar appears
  when preview points are pending; Learn triggers a confirmation popup before committing; Reset discards all staged
  changes
  - Pet talents - "Pet" button switches the frame to show your pet's talent trees; rebuilds automatically when you
  summon a new pet
  - Prerequisite lines - dependency arrows and branch lines drawn between talents, colour-coded by whether requirements
  are met
  - Compact mode - "Small" checkbox scales the frame to 80%.
  - Intercepts default frame - the microbar Talents button, hotkey, and any addon that calls the default talent frame
  all open ImprovedTalents instead
  - Persistent settings - compact mode preference saved between sessions via SavedVariables
